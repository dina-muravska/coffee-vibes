# ☕ Responsive Coffee Website

Modern responsive landing page for a coffee brand.  
This pet project focuses on clean UI design, smooth animations, and responsive layouts built with semantic HTML, modern CSS, and JavaScript.

The website showcases different coffee sections with engaging visuals and interactive elements to create a pleasant user experience.

---

## 🚀 Live Demo

- 🔗 Live Site: https://dina-muravska.github.io/coffee-vibes/
- 🔗 Repository: https://github.com/dina-muravska/coffee-vibes

---

## ✨ Features

- 📱 Fully Responsive Design – optimized for mobile, tablet, and desktop devices
- 🎯 Mobile First Approach – layout designed starting from small screens
- 🎬 Scroll Animations – elements appear with animations when scrolling
- 🧭 Smooth Navigation – smooth scrolling between sections
- 🎨 Modern UI – clean coffee-themed interface
- ⚡ Lightweight and Fast – built with pure HTML, CSS, and JavaScript

---

## 🛠 Technologies Used

- **HTML5**
  - Semantic markup
  - SEO-friendly structure

- **CSS3**
  - CSS Variables (Custom Properties)
  - Flexbox
  - CSS Grid
  - Responsive Media Queries

- **JavaScript (ES6+)**
  - Mobile menu functionality
  - Scroll interactions
  - UI behavior

- **ScrollReveal.js** – scroll-based animations  
- **Remix Icons** – scalable vector icons
